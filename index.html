Distributed Learning for Autonomous Vehicles in Communication-Constrained Environments

Nicole Cruz1, Jose Fuentes1, Leonardo Bobadilla1,
1 Florida International University
Paper

Abstract

Intelligent Autonomous Systems (IAS), which encompass Unmanned Aerial Systems (UAS), Unmanned Surface Vehicles (USV), and Unmanned Underwater Vehicles (UUV), are set to play a pivotal role across multiple domains. However, due to the demanding nature of their missions (e.g., disaster response and underwater exploration), common communication methods are often limited. This makes it essential to use efficient communication strategies for learning and decision-making. This paper introduces a novel approach, leveraging distributed machine learning algorithms along with data projection to achieve sublinear communication costs. Particularly, we incorporate Convolutional Neural Networks for feature extraction prior to projection, significantly enhancing efficiency and accuracy in autonomy tasks within bandwidth-constrained environments. This method uses the Johnson-Lindenstrauss transform for dimensionality reduction of both data and model parameters, facilitating efficient inter-agent communication by transmitting only projected model parameters. We further outline theoretical underpinnings showcasing the Johnson-Lindenstrauss transform's effectiveness in maintaining convergence properties for Online Gradient Descent algorithms. Through experimental validation using image recognition tasks relevant to maritime surveillance, our approach demonstrates substantial reductions in communication cost while preserving high model performance.

Methodology

Our distributed learning system for autonomous vehicles in communication-constrained environments operates through a three-stage cycle: Feature Projection, where each agent uses a Convolutional Neural Network (CNN) to extract compact feature representations from raw sensor data and applies the Johnson-Lindenstrauss (JL) transform to project both data and model parameters into a lower-dimensional space, preserving essential structure while drastically reducing communication costs; Surrogate Optimization, where each agent performs Online Gradient Descent (OGD) using the projected features and a reduced model architecture, maintaining theoretical convergence guarantees comparable to training on full-dimensional data; and Sequential Model Aggregation, where agents sequentially update and broadcast projected model parameters. The final agent performs an inverse projection to recover a high-fidelity model, enabling robust classification performance with sublinear communication overhead.

Distributed Learning Algorithm

The distributed learning framework enables multiple autonomous agents to collaboratively train a classification model while minimizing communication costs. Each agent extracts features from local sensor data using a shared CNN, projects these features to a lower dimension via a Johnson-Lindenstrauss transform, and performs Online Gradient Descent (OGD) on the compressed data. Updated model parameters are sequentially passed to the next agent, and the final agent reconstructs the full model for deployment. This approach ensures efficient, scalable learning with sublinear communication overhead.

Results

Experimental validation was conducted using both the CIFAR-10 benchmark dataset and a marine perception case study, demonstrating that the proposed distributed learning approach preserves classification accuracy while significantly reducing communication overhead. Three pre-trained CNN models—ResNet-50, VGG16, and EfficientNetB0—were used as feature extractors, followed by projection and training with logistic regression. Across 50 trials, models trained on projected data achieved comparable or superior performance to those trained on raw data, with reduced standard deviation in accuracy. These results confirm that dimensionality reduction via the Johnson-Lindenstrauss transform effectively supports scalable, reliable learning in bandwidth-constrained environments.
